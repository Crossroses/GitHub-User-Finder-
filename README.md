# GitHub-User-Finder-
GUI-приложение «GitHub User Finder» с использованием GitHub API, сохранением избранных пользователей и Git.
# GitHub User Finder

**Автор:** Коноплёва Влада
**Описание:** Простое GUI-приложение на Python для поиска пользователей GitHub с помощью API. Позволяет сохранять избранных пользователей в файл JSON.
**Технологии:** Python, Tkinter, Requests, GitHub API.

## Как использовать API

1. Введите логин пользователя GitHub в поле поиска.
2. Нажмите «Поиск» — приложение запросит данные через GitHub API.
3. Для добавления пользователя в избранное нажмите соответствующую кнопку.
4. Избранные пользователи сохраняются в файл `users.json`.

## Примеры использования

- Поиск существующего пользователя: введите `torvalds` (Линус Торвальдс).
- Поиск несуществующего пользователя: введите `nonexistentuser123` — появится ошибка.
- Добавление в избранное: после успешного поиска нажмите «Добавить в избранное».
